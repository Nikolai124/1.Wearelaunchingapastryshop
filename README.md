# Запуск кондитерской

сайт для кондитерской

## Как установить

Python3 должен быть уже установлен. 
Затем используйте `pip` (или `pip3`, есть конфликт с Python2) для установки зависимостей:
```
pip install -r requirements.txt
```

## Как Запустить проекта

Чтобы запустить проект для начала вам нужно в терминал вписать:
```
python main.py
```
после чего, зайдя в файл "template.html", справа внизу нажать кнопу "Go Live" и у вас откроется сайт
